# Environment Setup Guide

The `.env` file has been created with placeholder values. Follow these steps to complete the setup:

## 1. Database Setup (REQUIRED)

### Option A: Neon Database (Recommended - Free Tier)
1. Visit https://neon.tech
2. Create a free account
3. Create a new project
4. Copy the connection string
5. Replace the `DATABASE_URL` value in `.env`

### Option B: Other PostgreSQL Providers
- **Supabase**: https://supabase.com (Free tier available)
- **Railway**: https://railway.app (Free tier available)
- **ElephantSQL**: https://elephantsql.com (Free tier available)

## 2. AI API Keys (REQUIRED for AI features)

### Anthropic API Key (Primary AI engine)
1. Visit https://console.anthropic.com
2. Sign up and get your API key
3. Replace `ANTHROPIC_API_KEY` in `.env`

### OpenAI API Key (Optional)
1. Visit https://platform.openai.com
2. Get your API key
3. Replace `OPENAI_API_KEY` in `.env`

### Perplexity API Key (Optional)
1. Visit https://www.perplexity.ai
2. Get API access
3. Replace `PERPLEXITY_API_KEY` in `.env`

## 3. Security Secrets (ALREADY GENERATED)
✅ `SESSION_SECRET` - Auto-generated, ready to use
✅ `JWT_SECRET` - Auto-generated, ready to use

## 4. Optional Services

### SendGrid (for email notifications)
1. Visit https://sendgrid.com
2. Get API key
3. Replace `SENDGRID_API_KEY` in `.env`

### Stripe (for payments)
1. Visit https://stripe.com
2. Get test API keys
3. Replace `STRIPE_SECRET_KEY` and `VITE_STRIPE_PUBLIC_KEY` in `.env`

## Quick Start

After setting up your database and API keys:

```bash
# Install dependencies
npm install

# Push database schema
npm run db:push

# Build the application
npm run build

# Start the server
npm start
```

## Minimum Required for Testing

To get started quickly, you only need:
1. ✅ SESSION_SECRET (already generated)
2. ✅ JWT_SECRET (already generated)
3. ⚠️ DATABASE_URL (replace with real database)
4. ⚠️ ANTHROPIC_API_KEY (replace with real key)

The app will work with just these four variables configured!
